User Interface & User Experience Project

This project will be completed December 2024

This project was designed around solving a real world problem, putting user experience, and user interaction first. For this proect, I decided to prototype a task tracking app.

This project was broken down into several stages, each completed in 2-3 weeks.

Stage 1:

This stage was called "Empathize". Here I created a problem statement, researched demographics most likely to use this applicaation, and outlined a survery process. The survey was designed to learn how real world users interact with similar apps, what they like about them, what they dislike about them, and what they think could be done better. A total of five subjects were interviewed. From this stage, I generated empathy maps for each subject.

Stage 2:

This stage was called "Define". Here I created a persona to represent the general user of my application. I also created a buisness model for this application.

Stage 3:

This stage was called "Ideate". Here I created a mind map for my application, focusing on the things that are important to users gathered in the empathy maps during stage 1.

Stage 4:

This stage was called "Prototype". I started by creating several low fidelity prototype sketches, focusing on the features most imporant to the users. Then I created a high fidelity prototype using Figma, refining on the ideas from the low fidelity prototype.
